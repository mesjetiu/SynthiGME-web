# Experimentos / pruebas rápidas

Este directorio contiene prototipos aislados que no forman parte del build principal. Cada experimento vive en su propia subcarpeta y puede tener sus propias dependencias o instrucciones.

## Experimentos disponibles

- [`device-routing/`](device-routing/): pruebas de selección de dispositivos de entrada/salida con Web Audio (`getUserMedia`, `enumerateDevices`, `setSinkId`).

> Estos ejemplos son experimentales; su código puede cambiar sin previo aviso y no se incluye en `docs/` ni en la build principal.
